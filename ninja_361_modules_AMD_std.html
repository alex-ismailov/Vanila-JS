Кликни по экрану, после открой консоль

В стандарте AMD определена функция define() которой передаются:

  • Индентификатор вновь созданного модуля, необходим чтобы затребовать модлуль из других частей системы
  • Список зависимостей к текущему модулю
  • Фабричная функция инициализирующая модуль, который передается в качестве аргументов используемые модули
            define('MouseCounterModule', ['./dependencies/jquery-3.4.1.min.js'], $ => {
                let numClick = 0;
                const cilckHandle = () => {
                    console.log(`numClick: ${++numClick}`);
                };
            
                return {
                    countClick: () => {
                        $(document).on('click', cilckHandle);
                    }
                };
            });
